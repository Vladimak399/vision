# Review Workbench v2

Цель следующего этапа Review: превратить страницу проверки распознанных товаров в рабочий центр категорийного менеджера.

Уже есть в main:
- фильтры Review по статусу и candidates;
- safe bulk accept: массовое принятие не должно принимать размерный SKU, если OCR не распознал размер;
- предупреждение в подсказках каталога для size-risk candidates;
- быстрые ссылки на сессию и Excel exports.

Следующий безопасный инкремент:
- dashboard со счетчиками задач;
- отдельные queue-фильтры: AI candidates, риск размера, нет нашей цены, нет цены конкурента, низкая уверенность OCR;
- badges на карточках товаров;
- улучшенный active match block с явными предупреждениями;
- без миграций, без изменения OCR prompt, без auto-unmatched и без auto-accept AI candidates.
