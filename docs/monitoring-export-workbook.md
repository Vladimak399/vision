## Business rules

- A missing candidate is not the same as `–Э–µ—В –≤ –∞—Б—Б–Њ—А—В–Є–Љ–µ–љ—В–µ`.
- `–Э–µ –љ–∞–є–і–µ–љ–Њ` must contain only explicit `unmatched` rows.
- AI candidates are review hints, not accepted matches.
- Rows with `needs_review` are not final until manually confirmed.

1. Review `–С–µ–Ј –Ї–∞–љ–і–Є–і–∞—В–∞` first.
2. Review `AI candidates` and manually accept only safe matches.
3. Check `–Э–µ—В –љ–∞—И–µ–є —Ж–µ–љ—Л` before using price comparison.

5. Use `–Ь—Л –і–µ—И–µ–≤–ї–µ` for margin/opportunity review.
6. Export final results only after unresolved `needs_review` rows are handled.
